docker

基本使用命令:

docker里安装软件(gdb,gcc…)，使用docker commit -a"author" -m"commit log" containerid newimagename 提交镜像之后,可以使用docker save -o tag.tar newimagename 保存，创建镜像。

在另外的一个系统中需要使用这个镜像时，使用docker load -i tag.tar导入到另外一台主机
